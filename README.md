# Data Analyst Portfolio Website

A modern, professional portfolio website designed for Data Analysts. Built with HTML, CSS, and JavaScript.

## Features
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Premium Dark Mode**: Sophisticated aesthetic suitable for data professionals.
- **Sections**: Hero, About, Skills, Projects, Experience, Education, Testimonials, Contact.
- **Interactive Elements**: Smooth scrolling, mobile menu, and fade-in animations.

## How to Deploy to GitHub Pages

1. **Push to GitHub**:
   - Initialize a git repository if you haven't already: `git init`
   - Add files: `git add .`
   - Commit: `git commit -m "Initial commit"`
   - Push to your GitHub repository.

2. **Enable GitHub Pages**:
   - Go to your repository on GitHub.
   - Navigate to **Settings** > **Pages**.
   - Under **Source**, select `main` (or `master`) branch and `/ (root)` folder.
   - Click **Save**.

3. **View Your Site**:
   - GitHub will provide a URL (e.g., `https://yourusername.github.io/repo-name/`).
   - It may take a minute or two for the site to go live.

## Customization

- **Images**: Replace placeholder images in the `index.html` `src` attributes with your own.
- **Content**: Edit the text in `index.html` to reflect your actual experience and projects.
- **Colors**: Modify the CSS variables in `css/styles.css` (under `:root`) to change the color scheme.
